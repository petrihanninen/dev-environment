## Rainbow String 

Add rainbow effect to quoted string

![](https://raw.githubusercontent.com/wk-j/vscode-rainbow-string/master/images/Rainbow.png)

## Configuration

Add `rainbowString.extensions` to `settings.json`

```json
"rainbowString": {
    "extensions": [
        ".cs",
        ".csx",
        ".fs",
        ".fsx",
        ".js",
        ".jsx",
        ".ts",
        ".tsx",
        ".csproj",
        ".fsproj"
    ]
}
```
